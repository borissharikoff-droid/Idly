- Party System — create parties, roles (Tank/Healer/DPS), Party HUD, +5% XP buff
- Guild Hall — 10-level upgradeable building, collective donations, buff scaling (XP/Gold/Chest/Craft/Farm)

- Skill recognition — 80+ games, API/DB tools, video editors, note apps

- Arena zone cards — redesigned with power indicator (✓/~/✗)
- Profile — Quests tab for daily bounties + weekly challenges
- Craft/Cook — queued jobs list shown below active job

**Fixed:**

- Crafting XP rebalance (higher level = more XP, times cut 5-10×)
- Various minor bug fixes
