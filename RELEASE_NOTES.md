## What's new in v3.5.0

### New
- **Cooking system** — 16 recipes, 6 upgradeable instruments, mastery stars, Chef skill XP
- **DEF stat** — armor now grants Defense, reducing incoming damage in arena
- **Defense Potion** — permanent +1 DEF (max 50), drops from epic/legendary chests
- **Death Insurance** — craftable ward that prevents item loss on arena death
- **Dungeon Pass** — craftable pass consumed per auto-farm run
- **What's New modal** — patch notes shown on first launch after update
- **Polls** — new notification filter tab

### Fixed
- Auto-battle continues after boss chest claim without stopping
- DEF perk now contributes to Item Power calculation
- Item inspect modal now displays DEF stat on armor

### Improved
- Farm plots show exact rot % instead of vague "Rot risk"
- Seed Cabinet entrance animation (staggered slide-in)
- Profile tab shows orange badge for unclaimed achievement rewards
- Crafted body armor (Slime Shield → Troll Aegis) with DEF perks
- Plant combat buffs: Apples +1 DEF, Orchids +1 DEF, Crystal Root +2 DEF
- Warrior level 50 grants +2 DEF, level 80 grants +3 DEF
